# Refine Home Visual Parity

## Why

The homepage structure is now complete and closely matches the
information architecture of the source site.

However, visual differences still exist in typography, spacing,
card sizing, image treatment, section rhythm, and responsive
behavior.

These refinements should be handled independently from feature
development to keep changes focused and reviewable.

## What Changes

- Refine visual spacing across all home sections.
- Adjust typography scale and line-height.
- Improve desktop and mobile section rhythm.
- Refine image aspect ratios and cropping.
- Improve card sizing and visual hierarchy.
- Improve footer visual parity.
- Improve responsive behavior across 360 / 768 / 1280 / 1920 px.

## Impact

- No new functionality.
- No routing changes.
- No new dependencies.
- Affects only presentation and layout.
