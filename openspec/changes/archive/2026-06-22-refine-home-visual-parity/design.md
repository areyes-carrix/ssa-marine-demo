# Design Notes

## Goal

Match the visual appearance of ssamarine.mx as closely as practical
while preserving the existing architecture.

## Scope

Included:

- Hero
- About
- Services
- Why Choose Us
- Stats
- News
- Footer

Excluded:

- New routes
- New content
- Additional components
- Animation redesign

## Visual review process

For each section:

1. Compare against reference screenshots.
2. Measure spacing.
3. Measure image proportions.
4. Adjust typography.
5. Validate mobile and desktop layouts.

## Visual priorities

Highest priority:

1. Section spacing
2. Typography
3. Card sizing
4. Image framing

Lower priority:

- Shadows
- Border radius
- Transition timing

## Responsive targets

- 360px
- 768px
- 1280px
- 1920px
