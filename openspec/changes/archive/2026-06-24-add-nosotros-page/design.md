# Design: Nosotros Page

## Overview

The Nosotros page introduces the first internal content page outside the homepage.

The page should maintain the same visual language established by the homepage while matching the structure and content hierarchy of the original SSA Marine México About page.

## Goals

- Match the source website structure.
- Reuse the established design system.
- Keep content data-driven.
- Maintain responsive behavior across viewport sizes.
- Reuse existing extracted assets whenever possible.

## Route

```text
/nosotros
`
