# Design: Noticias Page

## Overview

The Noticias page introduces a dedicated news listing experience that mirrors the structure of the SSA Marine México website.

The page will reuse the existing site shell and PageHero component while introducing a data-driven news grid architecture.

This change also updates site navigation to better reflect the source website structure.

## Goals

- Match the overall structure of the source Noticias page.
- Reuse existing shared components.
- Maintain design consistency with the homepage and Nosotros page.
- Keep content data-driven.
- Preserve responsive behavior.
- Minimize duplicated implementation.
